Speech-2-Text API

Specs

The API is implemented as a POST request with two form parameters, file and parameters. The file should contain the audio file data, the optional parameters should be a JSON object with additional configuration. If no parameters are provided the default settings are used.
  • URL: /api/speech2text
  • curl -X POST -F "file=@[yourfilename]" http://HOST:PORT/api/speech2text
  • curl -X POST -F 'file=@[yourfilename]' -F 'parameters={"encoding": "LINEAR16"}' http://HOST:PORT/api/speech2text
    • encoding: LINEAR16, AMR, AMR_WB, FLAC, MULAW, OGG_PLUS, SPEEX_WITH_HEADER_BYTE
